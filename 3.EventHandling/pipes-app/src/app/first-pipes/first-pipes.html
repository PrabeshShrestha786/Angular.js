Built-in Pipes Demo

This page shows how Angular pipes transform values in the template.

DatePipe: {{ today | date:'fullDate' }}

CurrencyPipe: {{ amount | currency:'EUR':'symbol':'1.2-2' }}

PercentPipe: {{ percent | percent:'1.1-2' }}

LowercasePipe: {{ message | lowercase }}

UppercasePipe: {{ message | uppercase }}

JsonPipe (debug):

{{ user | json }}
